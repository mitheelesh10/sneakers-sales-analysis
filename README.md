# sneakers-sales-analysis
📊 Sneakers Sales Analysis Dashboard (Excel Project)

📌 Overview
This project analyzes sneakers and streetwear sales data using Microsoft Excel.  
The goal is to transform raw data into meaningful insights and build an interactive dashboard for better business decision-making.

---

🎯 Objectives
- Analyze overall sales performance  
- Identify top-performing brands and products  
- Understand customer preferences (gender & category)  
- Compare sales across different countries  
- Create an interactive Excel dashboard  

---

📁 Dataset Description
The dataset contains:
- Order Date  
- Product Name & Type (Sneakers, Hoodie, Cap, etc.)  
- Brand (Adidas, Off-White, Essentials, etc.)  
- Gender (Men, Women, Unisex)  
- Category (Streetwear, Casual, Limited Edition)  
- Country  
- Quantity Sold  
- Total Sales Amount  
- Payment Mode (UPI, Card, COD)  

---

🛠 Tools & Techniques Used
- Microsoft Excel  
- Data Cleaning  
- Pivot Tables  
- Pivot Charts  
- Dashboard Design  

---

📊 Dashboard Features
- 🌍 Sales by Country  
- 👟 Brand-wise Performance  
- 🧥 Category Analysis  
- 💰 Revenue & Quantity Overview  
- 🎛 Interactive Filters (Slicers)  

---
📈 Key Insights
- Sneakers and streetwear dominate total sales  
- Certain brands contribute the highest revenue  
- Sales distribution varies across countries  
- Cash on Delivery is widely used  
- Limited edition products generate higher revenue per unit
